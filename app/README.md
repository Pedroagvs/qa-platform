## Descrição do Projeto
A plataforma Quality Assurance pode ser utilizada em desktop ou web, sendo uma aplicação voltada para a gestão dos testes de usabilidade de aplicações. Isso possibilita que os QAs relatem os erros e melhorias encontradas durante os testes para os desenvolvedores. Dessa maneira, a plataforma fornece ao longo do tempo o histórico de testes de cada aplicação que foi salva, de acordo com o que os usuários forem criando.
